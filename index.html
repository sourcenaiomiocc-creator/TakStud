TakStud

A Escola na Palma da Mão

Apresentação do Projeto de Aplicativo Android

O Que é o TakStud?

  • Um aplicativo moderno para conectar professores e pais de forma simples e direta.
  • Acaba com o papelzinho na mochila: centraliza horários, avisos, trabalhos e provas no celular.
  • Feito para ser fácil de usar por qualquer pessoa, mesmo quem não tem muita intimidade com tecnologia.

Como foi Construído?

Usamos tecnologias confiáveis para garantir segurança e rapidez

Padrão Google

  • Kotlin: A linguagem oficial e moderna do Android.
  • Visual Moderno: Telas bonitas e fáceis de navegar.
  • Segurança: Seus dados protegidos na nuvem do Google.

Qualidade Técnica

  • Organização: O código é "arrumado" para facilitar atualizações futuras.
  • Sem Travamentos: Feito para rodar suave, mesmo em celulares mais simples.
  • Backup Automático: Salvamos tudo para você não perder nada.

Ferramentas Utilizadas

As melhores ferramentas para criar um app de qualidade

Android Studio

O software oficial de desenvolvimento Android - usado por milhões de desenvolvedores no mundo.

Claude AI

Inteligência Artificial avançada para escrever código mais rápido, com menos erros e melhor qualidade.

Git & GitHub

Controle de versão profissional - cada alteração do código é registrada e pode ser recuperada.

Para o Professor

Menos burocracia, mais ensino

Um painel de controle simples para gerenciar a turma em segundos.

  • Tarefas e Provas: Crie avisos de provas e trabalhos com poucos cliques.
  • Avisos Gerais: Envie recados importantes para todos os pais de uma vez.
  • Agenda: Organize o horário das aulas sem confusão.

Para os Pais

Acompanhe tudo de perto

Filtro Inteligente
Você vê apenas o que importa para a turma do seu filho. Sem bagunça.
Lição de Casa

Saiba exatamente quando são as provas e entregas de trabalhos.

Recados

Receba convites para reuniões e avisos da diretoria.

Horários

Confira a grade de aulas para ajudar a organizar a mochila.

Onde ficam as informações?

Usamos a tecnologia de nuvem do Google para garantir que nada se perca.

Tarefas

Aqui guardamos:

  • • Título do trabalho
  • • Descrição do que fazer
  • • Data de entrega
Avisos

Aqui guardamos:

  • • Título do aviso
  • • Mensagem completa
  • • Nível de urgência
Aulas

Aqui guardamos:

  • • Dia da semana
  • • Matéria (ex: Matemática)
  • • Horário de início/fim

Por que o TakStud é diferente?

Sem internet? Sem problemas!

O aplicativo continua funcionando mesmo se o Wi-Fi cair. Ele atualiza tudo assim que a internet voltar.

Atualização Instantânea

Se o professor postar um aviso agora, o celular dos pais apita no mesmo segundo.

Design Moderno

Telas limpas, letras grandes e botões fáceis de encontrar.
